<original_problem_statement>
The user's goal is to build a premium link and snippet manager app called Memor.ia.

**PRODUCT REQUIREMENTS (PRD) SUMMARY:**
- **App Concept:** An intelligent link and reference manager with a premium Apple-style aesthetic, dark mode, and smooth UX. The application is now pivoting to a mobile-first approach using React Native (Expo).
- **Core Functionality (Web & Mobile):**
    - **Link Management:** Save links, auto-scraped for title/image, add tags, mark as favorites.
    - **Clipboard Manager:** A dedicated section to save and manage plain text snippets/notes with emoji support and a quick copy button.
    - **Smart Clipboard:** An auto-save feature for copied content that activates for a 2-minute period.
    - **Folder System:** Organize links and notes into custom folders.
- **Monetization & Access Model (No User Login):**
    - **Free Trial:** New users get a 15-day free trial with full functionality.
    - **Premium Access:** After the trial, premium access is required. This is a one-time purchase (or will be).
    - **Influencer Codes:** Unique, single-use codes that grant permanent premium access. These are entered in the app's settings.
- **Admin Role:**
    - A separate, password-protected Admin Dashboard exists at the  web route.
    - Admins can create and manage influencer codes and view **anonymous, aggregated** application metrics (e.g., total installs, premium conversion rates). The dashboard must not contain any user-specific or personally identifiable information (PII).
- **Technical Stack:**
    - **Web App (Mostly complete, now a support/admin platform):** Next.js, Supabase, TailwindCSS.
    - **Mobile App (New focus):** React Native with Expo, targeting Android first, then iOS.
- **Backups & Sync (Critical for Mobile):**
    - Data (links, notes, premium status) must be backed up and restorable without an app-specific user account.
    - This must use the native OS cloud backup systems: iCloud for iOS and Android Backup Service/Google Cloud for Android.
- **Multi-language Support:** The app must support English, Portuguese, Spanish, and French.
